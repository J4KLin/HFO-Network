Compute features from HFO network centrality

Files:
'getCReP.m': 	
	-For a given patient, calculate critical resection percentage from ranked centrality scores and hforate.

'getPercentileRanks.m':
	-For a given patient, get the electrodes from the top X percentile of their electrode group (ie. resected volume, seizure onset zone, etc.) for each centrality type and hforate.

Requirements:
MATLAB		Version 9.10

Citation:
(In progress): "Network dynamics of interictal High Frequency Oscillations predict surgical outcome within the clinical workflow in refractory epilepsy"

Data Availability:
Relevant data can be found at: https://doi.org/10.7302/n9rt-sc45
